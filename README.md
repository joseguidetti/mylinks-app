## :bulb: Utilização
Utilize o [yarn](https://classic.yarnpkg.com/lang/en/) para iniciar o projeto

`$ yarn api` para subir na porta 3000 api.

`$ yarn start` para subir na porta 5000 o front.

> Para mais informações verifique o `package.json`
